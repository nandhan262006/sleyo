# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/nandhan262006/portfolio.git
```

Move into the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
